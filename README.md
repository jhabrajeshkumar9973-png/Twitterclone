Built a full-stack microblogging platform featuring user authentication, tweet creation, a dynamic home feed, and follower graphs.

Designed a relational MySQL database with optimized indexes to efficiently query and display chronological feeds from followed accounts.

Implemented server-side form validation, password hashing (bcrypt), and SQL injection prevention techniques using PDO/Prepared Statements.

Integrated AJAX/Fetch API to allow seamless, page-refresh-free interactions for liking and retweeting.
